# Smart-Cultural-Storyteller
Smart Cultural Storyteller

Smart Cultural Storyteller is a multimodal AI storytelling application built with Streamlit, Google Gemini APIs, Vertex AI Image Generation, and Google Text-to-Speech.
The system takes any uploaded story and transforms it into a rich, interactive cultural experience with intelligently generated summaries, visuals, character insights, narration, and alternate versions.
